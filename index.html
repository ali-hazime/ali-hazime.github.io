  • Super Capsule World

    Super Capsule World was the final project for my game design class. It is a side-scrolling platformer game that utilizes procedural generation to create the levels, spawn enemies, and power-ups. No two playthroughs will ever be the same! The game does not contain any art assets as the primary focus of the project and class was the programming side of game design.

    This was a project I worked on for my game design class. It is a three game cluster including a painting game, a bubble popper game, and a cube slicer game. Each game tracks and saves the player's high score which can be accessed from the main menu.
